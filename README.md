# 🇪🇨 Mini ERP Ecuador - Gestión de Ventas e Inventario

Sistema diseñado para digitalizar pequeños negocios ecuatorianos.

## 🚀 Tecnologías
* **Backend:** Java 17, Spring Boot 3, Spring Data JPA.
* **Frontend:** React.js, Tailwind CSS.
* **Base de Datos:** MySQL / PostgreSQL.
